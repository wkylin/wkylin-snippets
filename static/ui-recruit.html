  1. 1、在网站开发及改版的策划阶段，负责搭建出适合seo搜索的xhtml整站架构；

  1. 1、本科以上学历，对符合web标准的网站重构有丰富经验；
  2. 2、精通并能手写html/xhtml、css，对web标准和标签语义化有较深入理解；
  3. 3、了解常用浏览器，有效解决浏览器兼容问题；
  4. 4、了解javascript、jquery、ajax、actionscript等前端常用语言优先录用；
  5. 5、对前后端合作开发有一定的认知，对互联网产品有着良好的感觉，对交互体验有较为深入的了解；
  6. 6、准确把握前端发展方向，即时掌握前端新兴技术并灵活运用如html5、css3.0优先录用。
